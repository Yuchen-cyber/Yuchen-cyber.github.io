    A、FlowLayout B、GridLayout

    C、BordyLayout D、CardLayout

    原文链接：https://blog.csdn.net/weixin_41719737/article/details/86490339

  3. 下面哪个赋值语句是错误的？（）

    A、int i = 100 B、float f = 100.0

    C、String s = “Hello world” D、char c = ‘\123’

    选B

    float是单精度类型，double是双精度类型，如果不声明的话，默认小数都用double表示，所以如果要用floeat的话，则应该在其后加上f。所以正确的写法是 float f = 100.0f

    ————————————————
    原文链接：https://wenku.baidu.com/view/3a7781c0dd88d0d233d46af4.html
